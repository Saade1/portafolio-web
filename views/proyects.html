PROYECTOS

Aplicación Generador y Gestor de Cuestionarios

Este proyecto consistió en el desarrollo de una aplicación para la empresa High Tech que permite generar y gestionar cuestionarios de desempeño laboral. Su objetivo es facilitar a los administradores la creación y envío de encuestas, así como la recopilación y análisis de respuestas de manera rápida y eficiente.

Tecnologías Utilizadas:

  • Laravel
  • HTML
  • CSS
  • JavaScript
  • Bootstrap
  • PHP
